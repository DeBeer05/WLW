# WLW Front-End

A Flask-based front-end for WLW with a custom header, dropdown navigation, and dynamic card grid.

## Setup

1. Install dependencies:
   ```
pip install -r requirements.txt
   ```
2. Run the app:
   ```
python app.py
   ```

## Structure
- `app.py`: Flask app entry point
- `card_config.py`: Card configuration
- `templates/`: HTML templates
- `static/css/HEADER.CSS`: Header and dropdown styles
- `static/js/header.js`: Header JS
