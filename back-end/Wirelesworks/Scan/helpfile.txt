Usage:
    python Scan.py Options['-b', '-p', '-o', '-s', '-h']

    -b: BAUDRATE
        Set custom boudrate, by default 115200.

    -o: OUTPUT_FILE
        If defined after each scan the found devices will be printed to a file in json format.
        by default, no file will be writen.

    -s: SCANTIME
        Set a custom value for the time to scan in seconds, by default time is 5 seconds

    -h: HELP
        Show this help dialog.
